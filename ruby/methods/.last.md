# Rubyのlastメソッド

<br>
<br>

## lastメソッドとは
lastメソッドは配列の要素を取り出すときにその配列の中の最後の要素を取得するメソッド。
配列の要素がない場合は`nil`を返す。

  
 <br>
 
     
       
## lastメソッドの書き方

```ruby
配列名.last
配列名.last(引数)
```

<br>

```ruby
colors = ["red", "blue", "yello", "black"]
puts colors.last
puts colors.last(3)
```
